crear nuevo proyecto

git – version
Para revisar que Git esté instalado.
se abre una ventana de línea de comandos y se introduce la orden.

paso 1
creará la carpeta en el equipo llamada “git_adsi”

paso 2
git init
para empezar
estando ubicado en la carpeta a ser controlada.

paso 2
crear archivo “adsi.txt”.

paso 3
git add "nombre_del_archivo.ext" 
para añadir archivo a trabajar.

paso 4
git add .
para añadir todos los archivos al mismo tiempo.

paso 5
git status
revisar el estado de los cambios pendientes.

paso 6
git commit -m "mensaje"
git commit -m "version original"
Para confirmar los cambios.
se recomienda volver a revisar el estado (git status)

paso 7
git log
revisar el historial de cambios

paso 8
git log
revisar el historial de cambios

paso 9
realizar modificaciones al archivo inicial “adsi.txt” y volver a hacer todo de nuevo desde paso 3 o 4.

Nota: Se recomienda hacer este mismo ejercicio 10 veces seguidas.
      Dejar en comentarios el número de la versión
      y una frase describiendo el cambio, junto con un chiste.
      cuando terminen, por favor compatir sus "captures".
